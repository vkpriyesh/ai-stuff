## Why This Matters

Claude Code's system prompt is ~2000+ lines, battle-tested across millions of sessions. It represents Anthropic's best practices for:

- Controlling agent behavior
- Preventing harmful actions
- Ensuring output quality
- Managing tool use
- Balancing autonomy with safety
