# 02 - Custom Agents & Skills

Create custom agent types, skills, and personas for Claude Code without touching source code.

---

## Overview

Claude Code has three extensibility points you can use right now:

1. **Custom Agents** - Define new agent types in `.claude/agents/`
2. **Custom Skills** - Define slash commands in `.claude/skills/`
3. **System Prompt Overrides** - Modify Claude's behavior via CLI flags
